API RESTful

USUARIOS

GET /api/usuarios Obtener todos los usuarios.
POST /api/usuarios Añadir un usuario.
GET /api/usuarios/{usuario} Obtener un usuario.
UPDATE /api/usuarios/{id} Modificar un usuario.
DELETE /api/usuarios/{id} Eliminar un usuario.

CARROS

GET /api/carros Obtener todos los carros.
POST /api/carros Añadir un carro.
GET /api/carros/{placa} Obtener un carro.
UPDATE /api/carros/{id} Modificar un carro.
DELETE /api/carros/{id} Eliminar un carro.

RENTAS

GET /api/rentas Obtener todas las rentas.
POST /api/rentas Añadir una renta.
GET /api/rentas/{id} Obtener una renta.
UPDATE /api/rentas/{id} Modificar una renta.
DELETE /api/rentas/{id} Eliminar una renta.
